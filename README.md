# ✨ Beautiful Hover Effect Buttons

A curated collection of modern, smooth, and visually stunning button hover effects built purely with **HTML5** and **CSS3**. Zero JavaScript, fully responsive, and easy to drop into any web project.

---

## 🚀 Live Demo

See the buttons in action here: **https://malik25252007.github.io/Hover_Effects/**

---

## 🛠️ Features

*   **Pure HTML & CSS:** No bulky libraries or JavaScript dependencies.
*   **Modern Effects:** Includes glow, slide, border animation, magnetic-style visual cues, and 3D lifts.
*   **Customizable:** Built using CSS custom properties (variables) for effortless color and sizing tweaks.
*   **Accessible:** Designed with proper focus states and contrast ratios in mind.
